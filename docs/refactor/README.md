# Refactor Source Files

> [!NOTE]
> **Historical Artifacts**
> This directory contains the original Delphi source files (`.dfm` forms and `.pas` units) used as reference during the migration.
>
> They are kept here for:
> *   Checking business logic of the legacy system.
> *   Verifying field names and data types.
> *   Understanding original UI layouts.
>
> **They are not part of the build.**

## Directory Structure
*   `*.dfm`: Delphi Form definition (UI layout).
*   `pas/`: Delphi Pascal source code (Logic).
